## 📋 Your Deployment Checklist

### STEP 1: Install Node.js ⚙️
- [ ] Go to https://nodejs.org
- [ ] Download LTS version (Long Term Support)
- [ ] Run installer
- [ ] Restart your computer
- [ ] Open terminal, type `node --version` (should show a version)

### STEP 2: Prepare for Deployment 📦
- [ ] Open the webbsite folder
- [ ] Right-click → Open Git Bash Here (or regular terminal)
- [ ] Run: `npm install`
- [ ] Wait for it to finish (1-2 minutes)

### STEP 3: Test Locally 🧪
- [ ] Run: `npm start`
- [ ] Go to http://localhost:3000
- [ ] Test the contact form - fill it out
- [ ] Close server: `Ctrl + C`

### STEP 4: Push to GitHub 📤
- [ ] Go to https://github.com and create an account (if needed)
- [ ] Create new repository called "daylight-detailing"
- [ ] Follow GitHub's instructions to push your code
- [ ] Or use this simplified command:
  ```bash
  git config user.email "your@email.com"
  git config user.name "Your Name"
  git init
  git add .
  git commit -m "Initial commit"
  git branch -M main
  git remote add origin https://github.com/YOURUSERNAME/daylight-detailing
  git push -u origin main
  ```

### STEP 5: Deploy to Render 🚀 (EASIEST)
- [ ] Go to https://render.com
- [ ] Sign up with GitHub
- [ ] Click "Create New" → "Web Service"
- [ ] Select your "daylight-detailing" repository
- [ ] Settings:
  - Build Command: `npm install`
  - Start Command: `npm start`
  - Region: Sydney (or closest)
- [ ] Click "Create Web Service"
- [ ] Wait 2-3 minutes for deployment
- [ ] Your site is now LIVE! 🎉

**You'll get a URL like:** `https://daylight-detailing-xyz.onrender.com`

### STEP 6: Add Custom Domain (Optional) 🌐
- [ ] Buy domain from GoDaddy, NameCheap, or similar
- [ ] In Render dashboard → Settings → Custom Domain
- [ ] Add your domain
- [ ] Update DNS at your domain registrar (follow Render's instructions)
- [ ] Wait 24-48 hours for DNS to propagate
